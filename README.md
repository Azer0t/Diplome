# Вознесенська церква — 3D візуалізація

Інтерактивна 3D-візуалізація Вознесенської церкви (м. Ромни), пам'ятки архітектури кінця XVIII — початку XIX ст. Проєкт дозволяє детально оглянути будівлю, ознайомитися з її історією та архітектурними особливостями прямо у браузері.

## Про проєкт

Цей веб-застосунок створено для збереження та популяризації культурної спадщини. Користувачі можуть вільно переміщуватися навколо 3D-моделі храму, переглядати інформаційні анотації та фотогалерею.

### Основні можливості

*   **Інтерактивний 3D-огляд:** Делальна модель церкви з можливістю обертання та наближення.
*   **Режими керування:**
    *   **Orbit:** Обертання навколо об'єкта (стандартний режим).
    *   **Fly:** Вільний політ камерою (керування клавішами WASD).
*   **Анотації:** Інтерактивні точки на моделі з детальною інформацією про елементи архітектури.
*   **Галерея:** Перегляд архівних та сучасних зображень.
*   **Адаптивність:** Зручний інтерфейс як на комп'ютерах, так і на мобільних пристроях.

## Технології

*   HTML5
*   CSS3 (Modern Layouts, Variables)
*   JavaScript (ES6+)
*   Three.js — бібліотека для роботи з 3D-графікою у WebGL

## Як запустити

Оскільки проєкт використовує ES6 модулі, для коректної роботи необхідний локальний веб-сервер. Відкриття файлу `index.html` напряму у браузері може призвести до помилок CORS.

**Варіанти запуску:**

1.  **VS Code Live Server (Рекомендовано):**
    *   Відкрийте папку проєкту у Visual Studio Code.
    *   Встановіть розширення "Live Server".
    *   Натисніть "Go Live" у нижній панелі.

2.  **Node.js / npm:**
    *   Якщо у вас встановлено Node.js, можна використати `npx serve` або `http-server` у папці проєкту.

3.  **Python:**
    *   У командному рядку в папці проєкту виконайте: `python -m http.server`
    *   Відкрийте у браузері `http://localhost:8000`
